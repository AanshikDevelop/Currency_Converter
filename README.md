# 💱 Currency Converter

A sleek, intuitive **Flutter-based currency conversion app** designed to convert foreign exchange values in real time and serve as a practical tool for travelers, finance apps, or learning Flutter app architecture.

---

## 🧠 What This Project Is

**Currency Converter** is a mobile application built with Flutter and Dart that empowers users to:

- Convert between different global currencies
- See up-to-date conversion results instantly
- Experience responsive UI on Android, iOS, and web
- Learn scalable Flutter project structure

This project is ideal for beginners looking to deepen their Flutter skills and for portfolio showcases to demonstrate real-world app logic.

---

## 🚀 Core Features

✔️ **Live currency conversion** (logic implemented in app)  
✔️ Clean and responsive UI  
✔️ Multiple currency support  
✔️ Efficient state updates  
✔️ Works across platforms (Android, iOS, Web)  
✔️ Reactive input/output fields

🛠️ How to Run This Project
Prerequisites

Before you begin, make sure you have:

Flutter SDK installed

Dart enabled

Android Studio or VS Code

A connected emulator or device

Confirm setup with:

flutter doctor
Installation

Clone the repository

git clone https://github.com/AanshikDevelop/Currency_Converter.git

Navigate into the project

cd Currency_Converter

Fetch dependencies

flutter pub get

Run the app

flutter run
